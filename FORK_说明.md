## 如何 Fork 本项目

1. 打开 [Pixelclaw Office 仓库](https://github.com/FrankeyQu/Pixelclaw-Office)
2. 点击右上角 **Fork** 按钮
3. 选择目标账号，Fork 将复制到你的 GitHub 下
4. 克隆你的 Fork：`git clone https://github.com/<你的用户名>/Pixelclaw-Office.git`
